Auto keeps it updated from the live weather — a full-sun outdoor WBGT built from temperature, humidity, solar radiation, and wind (it reads as a shade value at night or under overcast). Turn Auto off to key in a meter reading. A meter is still the gold standard — always follow your camp medical SOP.

Heat Casualties

Heat stroke is a medical emergency. Hot skin + confusion, staggering, or collapse → call 911, cool the camper immediately (shade, water, ice to neck/armpits/groin), and notify camp medical. Log every heat case here as it happens.

Incident Report Log

each entry = 1 report
Log incidents by hand, or auto-import responses from your Google Form. Paste the form’s published CSV link (in the sheet: File → Share → Publish to web → the responses tab → Comma-separated values). New responses drop in automatically. Deleting an imported entry keeps it from coming back. The Medic column is auto-filled from the form’s “Name of Person Completing Report” field — the medic who submitted the report.

Field / PSG Broadcast

shows on the PSG phone page
Set this up once for the Platoon Sergeants’ phone page (psg.html). Whatever you type here appears on every PSG phone that’s connected to this board. The phones are read-only apart from Supply Requests — they can see the schedule, heat flag, and this announcement, and send supply requests (which land in the Supply Requests panel), but can’t change anything else on the board.
Hand this to the Platoon Sergeants
They scan the QR or tap the link, then pick their platoon — the sync link fills in automatically.
Hand this to the medics
Opens the medic dashboard (medic.html) with the sync link filled in. Medic requests carry no camper medical details.
Field documents
Links shown on the PSG Documents button. Host each file somewhere access-controlled (e.g. Google Drive, “anyone with the link”) and paste its share link here. These links sync privately over the sync link — they are not stored in this app’s public files, so keep sensitive docs (maps, plans) here rather than in the repo.

Live Sync

optional · shared board
Share one live board across computers. Paste your sync link (a Google Apps Script Web App URL) and press Connect — every device with the link shows the same board within a few seconds. Keep the link private: anyone with it can read and edit the board. See SYNC-SETUP.md in the project for the 5-minute setup.

Board backups

Automatic local snapshots of this board on this computer (taken as you work, before a reset, and before loading a file). If something gets wiped or a bad edit sticks, restore a recent one — restoring snapshots the current board first, so it's undoable. For off-computer safety, also use Save file now and then.
